java并发编程-6.栅栏

发表于2020-11-20|更新于2021-03-02|java并发

CyclicBarrier类

栅栏类似于闭锁，它能阻塞一组线程，直到某个事件发生。栅栏与闭锁的关键区别在于，所有线程必须都到达栅栏的位置时，才能继续执行。

  • 闭锁用于等待事件，栅栏用于等待其他线程。
  • 闭锁是一次性对象，一旦进入终止状态，就不能被重置。栅栏可以使一定数量的参与方反复地在栅栏位置汇集。如果所有线程都到达了栅栏位置，那么栅栏将打开，此时所有线程都被释放，而栅栏将被重置以便下次使用。
